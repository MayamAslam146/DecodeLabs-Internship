# DecodeLabs Data Science Virtual Internship 🚀

This repository documents my journey through the **Data Science Virtual Internship** offered by **DecodeLabs**. Each folder represents a weekly project covering essential data science skills — from data cleaning and exploratory data analysis (EDA) to feature engineering and machine learning pipelines.

---

## 📌 About the Internship

A hands-on, industry-oriented training program focused on building practical data science skills through real-world projects rather than just theory.
